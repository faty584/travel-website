## Overview

This project is a **responsive travel landing page** built with HTML and CSS. It’s tailored for desktop viewing and focuses on inspiring users with beautiful travel destinations while promoting the services of a global travel agency.


## 🚀 Project Features

### Section 1: Header

- Navigation bar with brand name and menu icon.
- Eye-catching visuals:
  - Blurry background image.
  - Decorative ellipse overlay.
  - Prominent travel-related image.
- Tagline: _“Travel around the world”_

### Section 2: Destination Highlights

- Features 6 destinations in card format:
  - Image of the location.
  - Destination name.
  - City and country (e.g., “Jawa Tengah, Indonesia”).
- Encourages exploration with a visually pleasing layout.

### Section 3: Why Choose Us

Three highlighted service cards:
1. **Best Guide, Best Experience** – Great trips with expert guides.
2. **Special Tickets** – Access to exclusive travel tickets.
3. **24/7 Support** – Always-on assistance for customer needs.

Includes call-to-action for more details.

### Section 4: Footer

- Links to:
  - Home
  - Destinations
  - Guide
  - Contact Us
- Social media icons: Instagram, Twitter, YouTube.
- Company slogan: _“The best travel agent in the world”_.

## Design and Styling

- **Fonts:**
  - _Gilda Display_ – Elegant headers.
  - _Urbanist_ – Clean body text.
- **Colors:**
  - Teal primary color: `#25A59E`
  - Soft grays and whites for contrast.
- **Responsive Design:**
  - Layout designed with desktop resolution in mind.
  - Clean spacing and alignment for clear readability.
